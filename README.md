## 🚀 TaskFlow – Frontend

O **TaskFlow Frontend** é a interface do usuário do sistema TaskFlow, responsável por consumir a API do backend e permitir o gerenciamento de tarefas de forma simples, segura e eficiente.

---

### 🎯 Objetivo

Simular uma aplicação real, integrando frontend e backend, com foco em:

- Autenticação de usuários
- Consumo de API REST
- Gerenciamento de estado
- Proteção de rotas
- Fluxo completo de tarefas (CRUD)

---

### ⚙️ Stack

- **React (Vite)**
- **JavaScript**
- **Axios**
- **React Router DOM**
- **Context API**

---

### 🔐 Funcionalidades

- Cadastro e login de usuários  
- Autenticação via JWT  
- Refresh automático de token  
- Proteção de rotas privadas  
- Logout seguro  

---

### 📋 Tarefas

- Criar tarefas  
- Listar tarefas do usuário  
- Atualizar tarefas  
- Remover tarefas  

---

### 🔗 Integração

- Comunicação com API REST  
- Envio automático de tokens nas requisições  
- Interceptação de erros com Axios  
- Logout automático em caso de falha de autenticação
